# Windows Terminal

-------------------------------------------------------------------------------
## Cygwin

- Cygwin setup
- Duplicate profile `Git Bash`
- Name: `Cygwin64 Terminal`
- Command line: `C:\\cygwin64\\bin\\bash.exe --login -i`
- Icon: `C:\\cygwin64\\Cygwin.ico`
- Do not call `mintty` (note; calling `mintty` creates a new window)
